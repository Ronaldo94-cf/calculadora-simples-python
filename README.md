📟 Calculadora em Python

Este é um projeto simples de uma calculadora desenvolvida em Python, que realiza operações matemáticas básicas como adição, subtração, multiplicação e divisão

⚙️ Funcionalidades

Somar dois números.

Subtrair dois números.

Multiplicar dois números.

Dividir dois números (com verificação para divisão por zero).



📝 Como Usar

Clone ou baixe o repositório:

Caso tenha o Git instalado, rode o comando:
git clone https://github.com/seu-usuario/calculadora-python.git



Entre na pasta do projeto:
cd calculadora-python



Execute o arquivo Python:
python calculadora.py



Interaja com a calculadora:

Escolha a operação que deseja realizar (1 - Somar, 2 - Subtrair, 3 - Multiplicar, 4 - Dividir).

Insira dois números para a operação.



💻 Exemplo de Execução


Ao executar o script, o programa pedirá para o usuário escolher a operação e fornecer dois números:
=== CALCULADORA ===
Selecione a operação:
1 - Somar
2 - Subtrair
3 - Multiplicar
4 - Dividir
Digite sua escolha (1/2/3/4): 1
Digite o primeiro número: 10
Digite o segundo número: 5
Resultado: 15.0




🛠️ Tecnologias Utilizadas

Python (qualquer versão acima de 3.6)



💡 Observações

Caso o usuário tente dividir por zero, a calculadora irá retornar uma mensagem de erro: "Erro: divisão por zero".

O programa continua pedindo entradas até que o código seja interrompido manualmente.
